## Dependencies for "editorjs" Drupal module.

This project provides next packages:

- [EditorJs core](https://github.com/codex-team/editor.js)
- [Code tool](https://github.com/editor-js/code)
- [Delimiter tool](https://github.com/editor-js/delimiter)
- [Header tool](https://github.com/editor-js/header)
- [Inline-code tool](https://github.com/editor-js/inline-code)
- [Link tool](https://github.com/editor-js/link)
- [List tool](https://github.com/editor-js/list)
- [Checklist tool](https://github.com/editor-js/checklist)
- [Table tool](https://github.com/editor-js/table)
- [Code lang tool](https://github.com/batkor/editorjs-code-lang.git)
- [Drupal image tool](https://github.com/batkor/editorjs-dimage.git)
- [Media image tool](https://github.com/batkor/editorjs-media-image.git)
- [Prism library (Only dependencies plugins)](https://github.com/PrismJS/prism)
